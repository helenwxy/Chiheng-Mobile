呼吸道疾病

1. 慢性阻塞性肺病（慢阻肺）
2. 哮喘
3. 流行性感冒
4. 肺炎

1. 慢性阻塞性肺病

慢阻肺是由于长期的肺部损伤（如吸烟）导致的呼吸困难。 随着时间的拖延， 呼吸困难越发严重。 肺部损伤是不可逆的。一共有两种慢阻肺：

  • 慢性气管炎：输送空气进入肺部的管道发生炎症， 导致大量粘液产生。这种状况导致气管狭窄甚至阻塞， 导致呼吸困难。
  • 肺气肿：肺泡被破坏导致不能伸缩。 进出肺部的气体减少，产生呼吸困难的感觉

1.1慢阻肺的症状

  • 长期（慢性）咳嗽，长于3个月
  • 咳嗽时有黏液
  • 呼吸困难
  • 肺部感染（流感，急性气管炎，肺炎）病程长于其他人
  • 啸鸣音（呼吸时有哨音）
  • 感觉疲乏
  • 体重下降（非刻意为之）

    • 1.2治疗/疾病控制方式

  • 戒烟
  • 呼吸技术控制：吹笛式呼吸法
  • 能量管理：合理规划每天的活动，了解个人活动极限，保持有规律的活动
  • 药物使用：可包含处方吸入型药物、激素类药物、抗生素、流感和肺炎疫苗
  • 氧气疗法（注，氧气附近不可点火，点烟，可引起着火或爆炸。不可再易燃物附近放置氧气瓶。）
  • 正确的身体姿势：上身前倾以减轻横膈肌压力

    • 1.3慢阻肺急性发作症状

  • 呼吸比平时更困难
  • 痰比平时更黄，呈绿或褐色
  • 痰量增加，更粘稠
  • 发烧
  • 感冒症状，如头痛，流鼻涕或咽喉痛
  • 关节肿胀
  • 感觉疲乏或不舒服
  • 严重的急性发作需立即呼叫救护车，症状包括：胸痛，口唇和指尖发紫，意识混乱，思路不清晰或急躁，昏昏沉沉，呼吸机端困难

    • 1.4预防慢阻肺急性发作

  • 健康饮食，规律锻炼，充足睡眠
  • 注意手部卫生（勤洗手）
  • 避免触摸眼、嘴、鼻、口
  • 远离其他病人
  • 按医嘱服药
  • 合理规划日常活动
  • 每年接受流感疫苗注射，每5到10年接受肺炎疫苗注射
  • 避免使慢阻肺恶化的因素


    • 2. 哮喘

    哮喘是当肺中的气管受到刺激，变得狭窄、肿胀，同时黏液产生，进而造成呼吸困难。哮喘无法根治，但是可以得到有效的控制。哮喘分为轻微、 中等、和重度哮喘。重度哮喘时常会干扰到日常生活，哮喘发作时可危及生命。

    2.1哮喘发作症状

    有些有哮喘的人可能会频繁哮喘发作，有些人的症状仅仅在特定请款下发声，但不是一成不变的。症状通常伴随呼吸系统感染而 加重哮喘发作的症状包括：
  • 呼吸困难，剧烈咳嗽
  • 胸部紧张感或疼痛感
  • 由于呼吸困难导致的失眠
  • 呼吸时有啸鸣声（类似哨声）

    • 2.2哮喘吸入药剂

  • 控制剂/预防剂：降低气管炎症
  • 缓解剂/急救剂：快速缓解症状（咳嗽，啸鸣，呼吸困难）
  • 注：需留心观察使用缓解剂的频率

    • 2.3护理人员须知：何时就医

    当患者出线以下症状时，需呼叫救护车，并在此期间继续使用急救剂
  • 呼吸极度困难
  • 急救剂无效
  • 说话困难，不能完整说完一个句子
  • 急救剂无效
  • 鼻孔大张
  • 皮肤苍白，发灰，出汗
  • 嘴唇及指甲发蓝紫色
  • 体力疲乏、神情淡漠
  • 失去意识或昏倒


    • 3. 流行性感冒（流感）

    由流行性感冒病毒引起的上呼吸道感染，通过飞沫传染，常见于秋冬天（11-5月）。流感会增加其他感染的风险，流感并发症可能会危及生命。

    3.1 流感症状

  • 发烧，头痛，肌肉酸痛，流鼻涕，喉咙痛，咳嗽，极度劳累
  • 儿童会出现恶心，呕吐或腹泻的症状
  • 流感症状会在初次接触到流感病毒的一到四天后开始发作，症状持续七至十天，但咳嗽和体虚症状会持续一到两週

    • 3.2 预防流感

  • 打流感疫苗
  • 经常洗手
  • 用过的纸巾要立即扔进垃圾桶
  • 咳嗽和打喷嚏时，用衣袖或领口盖住口鼻，而不是直接用手
  • 患病期间要留在家中
  • 经常触摸的物品需经常清洁，消毒

    • 3.3 在家中治疗流感

  • 充分休息，尽量留在家中
  • 多喝水以弥补发烧造成的液体流失
  • 避免吸烟
  • 吸入蒸汽以缓解鼻塞症状
  • 服用抗流感或抗病毒的药物
  • 可服用咳嗽或感冒药来减轻流感症状 （泰诺，雅维）

    • 3.4 何时就医

  • 病状加重
  • 出现气短，呼吸困难
  • 胸痛
  • 脱水迹象
  • 高烧不退
  • 神志不清，意识混乱


    • 4. 肺炎

    肺炎是由病原体通过呼吸进入肺部导致的肺部感染。在感冒或流感后更容易患上肺炎。大部分患者可在家治疗，但如果症状加重，需主要治疗

    4.1 肺炎症状

  • 咳嗽并咳出铁锈色或绿色黏液，有时带血
  • 发烧超过38度
  • 寒颤
  • 呼吸急促或呼吸困难
  • 胸痛，当咳嗽或吸气时加重
  • 心率加快/li>
  • 感觉疲惫或虚弱
  • 高龄患者可伴有精神症状，如思维混乱，糊涂

    • 4.3 在家中治疗肺炎

  • 多喝水，水可稀释粘痰，使之易于排出
  • 注意休息
  • 在需要时服用退烧和止咳药
  • 服用抗生素一定要谨遵医嘱，需持续服药到疗程结束。否则，感染可能复发
  • 即使几天内感觉好转，2周后也要约见医生.

    • 4.4 何时就医

    如出现以下症状，需立即去医院急诊
    • 呼吸困难,咳嗽不止，胸部或周围感觉发紧
    • 由于喘不上气而讲话困难
    • 嘴唇和指尖变成紫色
    • 挤压性胸痛
    • 咳出鲜红色血
    • 站起来或坐起来时头晕
    以下症状需就诊（非急诊）
    • 咳出带有血点的黏液
    • 咳出带有铁锈色，黄或绿色的粘痰，超过2天
    • 咳嗽导致呕吐
    • 咳嗽症状持续四周以上
    • 发烧超过38.5度，并伴随寒颤
    • 感觉呼吸困难，呼吸频率高于正常
    • 呼吸有哮鸣音
    • 新发的胸痛，咳嗽和深呼吸时加重
